# pyenv

## Setup 
1. *Create A Virtual Env*
```
python -m venv myenv
```

2. *Activate Virtual Env*
```
source myenv/bin/activate
```
_for linux only_

3. *Install libs fron requirements.txt*
```
pip install -r requirements.txt
```

4. *Deactivate env*
```
deactivate
```
